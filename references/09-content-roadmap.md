# 09 · Plan de contenido 12 meses

## Modelo de producción (híbrido)
El **experto dicta** outline + datos clave (30 min de audio o doc breve) → **redactor SEO** transcribe a copy → **el experto revisa y firma** antes de publicar (E-E-A-T). Costo orientativo: $80-150 pieza larga, $40-80 pieza corta.
- AioTech: output sostenible **4-6 piezas/mes**.
- Inspira (la experta firma todo, YMYL): **2-3 piezas/mes**, más conservador.

## Roles
| Rol | Responsable | Tiempo/mes |
|---|---|---|
| Strategy + briefs | Líder técnico/experto | 8-12 h |
| Drafting | Redactor freelance SEO | (externo) |
| Technical review | Experto del área | 4-6 h |
| Publishing + on-page | Dev/SEO | 6-8 h |
| Casos (entrevista+redacción) | Experto + redactor | 8 h por caso |
| Promoción | Dev + experto | 4 h |

## Roadmap trimestral
- **Q1 (M1-3) FOUNDATION** — páginas core + hubs de servicio + GBP + foundation files (robots/sitemap/llms.txt). Luego spokes. Luego sectores + primera ubicación + primer caso + Pillar 1 + 2 cluster posts.
- **Q2 (M4-6) LOCAL + SECTOR EXPANSION** — ubicaciones + casos + Pillars 2-3 + inicio LinkedIn long-form.
- **Q3 (M7-9) AUTHORITY BUILDING** — pillars finales + clusters + casos + refresh de Q1 con data GSC + guest posts + Q&A foros + push reviews + webinar.
- **Q4 (M10-12) OPTIMIZATION + SCALE** — auditoría completa + refresh top 10 páginas + A/B en CTA + campaña link building dirigida + cierre con reporte + plan año 2.

## Estructura común de un hub
H1 + subhead + **TL;DR citable** + "¿Para quién es?" + señales/bullets + "Cómo trabajo/operamos" (sin promesas) + modalidad/cobertura + FAQ + enlaces internos + CTA + WhatsApp prellenado.
